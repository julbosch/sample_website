Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header Make a header for the page
div divider separate page into referencable chunks
ol ordered list Create an ordered list of things
ul unordered list Create an unordered list of things
li list item An item in the list

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong looking text <strong>technical sophistication</em> technical sophistication
a link make a link to another page or location <a href="https://www.learnenough.com/html-tutorial">Learn Enough HTML Tutorial</a> Learn Enough HTML Tutorial
img image embed an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code format as monospaced <code>text here</code> text here
span span used to apply styling to text within <span>text here</span> text here

Document Tags

Tag Name Purpose
html html Tells the browser it is an HTML doc
head head Area of the HTML doc to put non-content stuff like styling, scripts etc.
body body Area of the document where the content goes
title title The title of the HTML page, displays in the tab on the broswer and is required for HTML5
meta meta used to set the character set the page will recognize and maybe other stuff as well?
